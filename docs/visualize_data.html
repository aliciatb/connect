Visualize with ggplot2

ggplot2 is a package for visualizing data in R. Think of it as a “grammar of graphics”. With ggplot2, one can create all manner of visualizations. We we also explore gganimate, a package for bringing a visual to life.

Visually Display the Number of Pups and Non-Pups by Site

The Grammar of Graphics offers up three key elements when creating a visualization: * data, or what will be visualized * geom, or how to display the data * aes, or the aesthetics of the display

The structure:

ggplot( data to be used ) + geom to be used ( mapping = aes( to be used like what is x, y, etc) )

ggplot(data = region_counts) +
  geom_col(mapping = aes(x = Region, y = Non.Pup))

Trees! (in Edmonton)

Reading in the Trees from Edmonton’s Open Data site ()

url <- "https://data.edmonton.ca/resource/93cp-z7sw.json?$limit=10000"  ## Just 10,000 records to make things a little faster
trees <- read.socrata(url)

Are larger trees in better or worse condition?

ggplot(trees) +
  geom_point(aes(diameter_breast_height,condition_percent))

Is a certain genus of tree suffering more than others?

ggplot(trees) +
  geom_count(aes(genus,condition_percent))
